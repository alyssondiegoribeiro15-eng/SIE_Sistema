# Sistema de Estoque Industrial (SIE)

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- MySQL
- NetBeans
- GitHub

## Desenvolvedor
- Alysson Diego Ribeiro

## Objetivo do Sistema
O sistema tem como objetivo controlar o estoque de um almoxarifado, permitindo o cadastro de materiais, movimentação de entrada e saída, e consulta de estoque.

## Funcionalidades
- Cadastro de materiais
- Movimentação de estoque (entrada e saída)
- Consulta de estoque
- Controle de saldo atualizado
